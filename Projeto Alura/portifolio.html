Cardápio de comida grega: Prato de Poseidon

Este projeto é um cardápio de comida grega que apresenta uma lista de alimentos, incluindo informações sobre os pratos, valores, composição e etc. A página é estilizada com CSS para uma visualização agradável.

A estrutura do site é baseada em HTML, que define a semântica e o layout do conteúdo, enquanto o CSS é usado para estilizar a página visualmente, incluindo cores, tipografia e a disposição dos elementos.

O HTML organiza o conteúdo em um cabeçalho com o título do site, seguido por uma divisão principal (.container) que apresenta os pratos do restaurante, e informações para compra, além de suas categorias. Cada categoria de prato é destacada em seu próprio contêiner (.prato), mostrando uma imagem do prato servido, detalhes do prato exibido e informações para compra. O design responsivo é garantido pelo uso de uma meta tag viewport e um layout flexível que se adapta a diferentes tamanhos de tela.

O CSS personaliza o visual do site, usando variáveis para cores. O uso de HTML e CSS é importante pois assim é possível criar um site acessível e esteticamente agradável sem a necessidade de scripts complexos, com foco na usabilidade e na experiência da pessoa usuária. A escolha da tipografia e do esquema de cores contribui para a atmosfera acolhedora do site, incentivando a exploração da coleção pratos e bebidas.

Fotografia urbana: Ruas em foco

O projeto consiste em uma aplicação web interativa, desenvolvida com HTML, CSS, e JavaScript, que mostra fotos artisticas de fotografos ao redor do mundo. O objetivo é explorar o comércio artistico e abrir novas oportunidades para artistas iniciantes.

O HTML estrutura o conteúdo da página, dividindo-o em seções como a tela inicial, exposições, galeria, locais e etc. O CSS é utilizado para estilizar a página, empregando um esquema de cores futurista e layout responsivo para melhorar a experiência do usuário. JavaScript é usado para adicionar interatividade ao site, manipulando o DOM para atualizar o conteúdo da página conforme a interação do usuário.

O HTML organiza o conteúdo em um cabeçalho com o título do site, seguido por uma divisão principal (.container) que apresenta o propósito do site e a coleção de fotos em uma seção flexível (.galeria). Cada foto é destacada em seu próprio contêiner (.foto), mostrando uma imagem, detalhes da foto e informaçõed o fotografo. O design responsivo é garantido pelo uso de uma meta tag viewport e um layout flexível que se adapta a diferentes tamanhos de tela.

Os scripts JavaScript incluem um carrossel de imagens, um FAQ e um menu hamburgue do lado esquedo do site na versão mobile. Este design modular facilita a manutenção e a expansão futura do site. A importância das ferramentas utilizadas (HTML, CSS, JavaScript) reside na criação de uma experiência interativa que não apenas entretém, mas também ensina os usuários sobre a complexidade ética e social da arte, incentivando a reflexão crítica sobre o papel da mesma em nossas vidas.
